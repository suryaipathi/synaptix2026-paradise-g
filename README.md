# Adaptive Online Assessment Platform

## 📌 Overview

A web-based adaptive assessment system that dynamically adjusts question difficulty based on student performance. The platform includes authentication, timed quizzes, performance analytics, and subject-wise mastery evaluation.

---

## 🚀 Features

* Student login authentication
* Adaptive difficulty algorithm
* Multi-subject quiz system
* Timer-based assessment
* Performance analytics dashboard
* Strongest & weakest subject detection
* Interactive UI

## ⚙️ How It Works

1. User logs in using student ID and password
2. Quiz begins with medium difficulty
3. Difficulty increases/decreases based on performance
4. Subject-wise accuracy is tracked
5. Final report displays mastery levels and analytics

---

## 🧠 Adaptive Algorithm

* 2 consecutive correct answers → Increase difficulty
* 2 consecutive wrong answers → Decrease difficulty
* Difficulty range: Easy → Medium → Hard
